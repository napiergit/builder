# Cookie Cutter MCP Template

Standard template for generating consistent MCP servers with OAuth integration.

## Features

- OAuth authentication flow
- Environment variable management  
- Standard MCP server structure
- Testing framework
- Deployment configuration

## Template Variables

- `{{platform_name}}`: Name of the platform (e.g., "gmail", "github")
- `{{client_id}}`: OAuth Client ID
- `{{client_secret}}`: OAuth Client Secret  
- `{{redirect_url}}`: OAuth Redirect URL
- `{{server_description}}`: Description of MCP server functionality
